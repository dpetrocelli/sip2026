Práctica 0 — Prerrequisitos

Guía paso a paso para instalar k3s (o k3d) y dar los primeros pasos con Kubernetes. Requisito obligatorio para entregar la Parte 2.

TP 0

Prerrequisitos: k3s + kubectl

Instalación de k3s en Linux/WSL o k3d en macOS/Windows, comandos básicos de kubectl, hello-world con nginx, y conceptos mínimos (Pod, Job, CronJob, ConfigMap, PVC) para encarar el Hit #8.

Requisito para: Parte 2 (02/05/2026) → Ver guía

Práctica I — Automatización Web

Pruebas E2E multi-browser sobre MercadoLibre con Selenium WebDriver.

TP 1 · PARTE 1

Selenium + MercadoLibre — bases

Hits #1–#3: setup de Selenium WebDriver, Browser Factory para Chrome y Firefox, y aplicación de filtros vía DOM (nuevo + tienda oficial) con captura de screenshots.

Entrega: 25/04/2026 → Ver TP
TP 1 · PARTE 2

JSON, Tests, Docker, k3s y CI/CD

Hits #4–#9: extracción estructurada a JSON, scraper resiliente, tests con cobertura ≥ 70 %, Dockerfile multi-browser, pre-commit hooks, despliegue en k3s/k3d (Job + CronJob + ConfigMap + PVC), ADRs y bonus (paginación, POM, Helm).

Entrega: 02/05/2026 → Ver TP

Práctica II — Observabilidad (4 partes)

Logging centralizado y trazabilidad del scraper sobre k3s, recorriendo tres stacks (Loki/Promtail/Alloy/Grafana, EFK y OpenTelemetry) y cerrando con una comparativa arquitectónica y un ADR magisterial.

TP 2 · PARTE 1

Loki + Promtail/Alloy + Grafana

Logging centralizado del scraper en k3s: recolección de logs con Promtail/Alloy, indexación en Loki, exploración y dashboards en Grafana. Aplica conceptos de la Parte 2 del TP 1 (Job/CronJob, ConfigMap, PVC).

Entrega: 05/05/2026 → Ver TP
TP 2 · PARTE 2

EFK: Elasticsearch + Fluent Bit + Kibana

Logging centralizado con el stack EFK: Fluent Bit como recolector liviano, Elasticsearch como motor de indexación full-text y Kibana para exploración y dashboards. Comparación operativa frente a Loki.

Entrega: 05/05/2026 → Ver TP
TP 2 · PARTE 3

OpenTelemetry: Collector + SDK + multi-backend

Instrumentación con OpenTelemetry: SDK en el scraper, OpenTelemetry Collector como pipeline unificado y exportación a múltiples backends (Loki, Elastic, Tempo/Jaeger). Logs, métricas y trazas correlacionadas.

Entrega: 09/05/2026 → Ver TP
TP 2 · PARTE 4

Cierre: comparativa y ADR magisterial

Cierre de la Práctica II: comparativa cuantitativa y cualitativa de los tres stacks, criterios de decisión arquitectónica y redacción de un ADR magisterial que justifique la elección para un escenario productivo.

Entrega: 09/05/2026 → Ver TP
